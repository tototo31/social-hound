# social-hound
social-hound OSINT · RECON · ASSET AUDIT intelligence tool
How to open |
Linux :
1. Create venv in your project folder
python3 -m venv venv

2. Activate it
source venv/bin/activate

3. Install packages
pip install requests beautifulsoup4 dnspython rich pyfiglet python-whois

4. Give acces to Tool
chmod +x social-hound.py

5. Run Tool
python social-hound.py

Windows : 
1. Install packages
pip install requests beautifulsoup4 dnspython rich pyfiglet python-whois

2. Run 
Double click or copy path and paste into your terminal
